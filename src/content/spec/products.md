## 一、爆款标题 (10条)

### 1. 数字悬念型
```
帮我写 5 个小红书标题，主题是【你的产品/话题】。
要求：包含具体数字（比如"3个""5步""7天"），制造悬念感，让人想点进来。
风格：像朋友分享，不要像广告。
```

### 2. 痛点解决型
```
帮我写 5 个小红书标题，主题是帮【目标人群】解决【具体问题】。
要求：标题前半句说出痛点，后半句暗示有解决方案。
例："化妆卡粉？这3个方法让我再也卡过"
```

### 3. 对比反差型
```
帮我写 5 个小红书标题，用"以前...现在..."或"别人...我..."的对比结构。
主题：【你的产品/话题】
要求：突出反差的戏剧性，让人好奇。
```

### 4. 省钱攻略型
```
帮我写 5 个小红书标题，主题是【你的产品】省钱攻略。
要求：包含钱数（比如"省了500块""才花了30"），强调性价比。
```

### 5. 懒人救星型
```
帮我写 5 个小红书标题，主题是【你的产品】。
要求：强调"懒人""不用动脑""有手就行"等低门槛关键词。
```

### 6. 避坑指南型
```
帮我写 5 个小红书标题，主题是【你所在的领域】的避坑经验。
要求：包含"别踩""避雷""后悔没早"等警示词。
```

### 7. 真实体验型
```
帮我写 5 个小红书标题，分享我使用【你的产品】的真实经历。
要求：像日记一样自然，不说"推荐""必买"。
```

### 8. 教程步骤型
```
帮我写 5 个小红书标题，教别人怎么做【你的技能】。
要求：标题里体现步骤感，比如"从0到1""手把手""一步步"。
```

### 9. 情绪共鸣型
```
帮我写 5 个小红书标题，表达【目标人群】的共同感受。
要求：用感叹号、口语化表达，让人一看就觉得"我也是！"
```

### 10. 结果展示型
```
帮我写 5 个小红书标题，展示使用【你的产品/方法】前后的变化。
要求：有视觉冲击力，比如"一个月前vs一个月后""用了和没用的差别"。
```

## 二、正文结构 (10条)

### 11. 清单体笔记
```
帮我写一篇小红书清单体笔记，主题是【你的主题】。
结构：
- 一句话开头（点出痛点或制造好奇）
- 列出 3-5 个要点，每个要点 2-3 句话
- 结尾引导互动（问一个问题或求点赞收藏）
语气：像闺蜜聊天，说人话，别用书面语。
```

### 12. 故事型笔记
```
帮我写一篇小红书笔记，用讲故事的方式推荐【你的产品/方法】。
结构：
- 开头：我以前...（描述一个具体的糟糕场景）
- 转折：直到我发现了/试了...（引入你的产品）
- 结果：现在...（描述改善后的状态）
- 结尾：你也可以试试，方法放在评论区
要求：细节要真实，像朋友圈而非广告。
```

### 13. 评测对比型
```
帮我写一篇小红书笔记，对比【产品A】和【产品B】。
结构：
- 一句话说结论
- 分别列出A和B的优缺点（3点即可）
- 我选择X的原因
- 你用过哪个？评论区说说
要求：客观，不要一面倒。优缺点都要有。
```

### 14. 干货教程型
```
帮我写一篇小红书教程笔记，教别人怎么【做某件事】。
结构：
- 为什么需要学这个（10字以内）
- 准备材料（列出3样以内）
- 步骤1、2、3（每一步配一句说明）
- 成品展示或效果对比
要求：每一步都要普通人能看懂，少用术语。
```

### 15. 好物推荐型
```
帮我写一篇小红书好物推荐笔记，推荐【你的产品】。
结构：
- 它解决了我什么痛点
- 它好在哪里（3个优点）
- 适合谁用
- 在哪买/多少钱
要求：优缺点都要提到，客观真实。
```

### 16. 合集整理型
```
帮我写一篇小红书合集笔记，主题是【你领域的Top5推荐】。
结构：
- 开头一句话说明这个合集的价值
- 5个推荐，每个50字以内，含推荐理由
- 结尾总结（一句最推荐哪个）
```

### 17. 问答互动型
```
帮我写一篇小红书互动笔记，发起一个跟【你的话题】相关的讨论。
结构：
- 抛出问题（如"你们觉得...""有人跟我一样...吗？"）
- 先说我自己的答案或经历
- 引导评论：你呢？快来聊聊
```

### 18. 日常vlog型
```
帮我写一篇小红书日常分享，记录我【做某件事/用某产品】的一天。
结构：
- 时间线串起来（早上...中午...晚上...）
- 每个时间点2-3句话
- 配上心情和感受
要求：真实感最重要，像日记。
```

### 19. 翻车避雷型
```
帮我写一篇小红书"避雷"笔记，分享我在【某领域】踩过的坑。
结构：
- 我翻了什么车（具体事件）
- 为什么踩坑（原因分析）
- 后来怎么解决的
- 总结：大家千万别...
要求：自嘲幽默的口吻，不要高高在上。
```

### 20. 一键生成型
```
帮我写一篇完整的小红书笔记，主题是【你的主题】。
必需元素：
- 一个带数字或情绪的标题
- 3-4段正文（每段不超过3句话）
- 3-5个相关标签
- 结尾互动句
- emoji适量点缀（3-5个）
```

## 三、高级技巧 (10条)

### 21. 封面文案优化
```
我准备发一篇小红书笔记，主题是【你的主题】。帮我设计3个不同的封面文案方案。
每个方案包含：
- 封面大字（6字以内，有冲击力）
- 副标题（10字以内，补充说明）
- 配图建议（什么类型的图片）
```

### 22. 评论互动话术
```
帮我写5条回复评论的模板，用于我在小红书推荐【你的产品】的笔记下。
要自然、亲切，引导更多互动。不要机械的"谢谢关注"。
```

### 23. 标签策略
```
帮我想15个小红书标签，用于关于【你的主题】的笔记。
要求：
- 5个热门大标签（流量大）
- 5个精准小标签（竞争小）
- 5个长尾标签（搜索意图明确）
```

### 24. 发布时间建议
```
告诉我在小红书发关于【你的话题】的笔记，最佳发布时间是什么？为什么？不同时间段适合发什么类型的笔记？
```

### 25. 系列化策略
```
帮我设计一个3篇系列笔记的主题和内容大纲，围绕【你的领域】。
要求：每篇有独立主题，但又互相关联，引导读者追更。
```

### 26. 蹭热点技巧
```
现在是【某节日/事件】，帮我设计一个跟【你的产品/领域】结合的小红书选题。
要求：自然嫁接，不生硬，让人觉得有创意而非硬蹭。
```

### 27. A/B测试方案
```
针对【你的主题】，帮我设计2个不同风格的小红书笔记方案：
- 方案A：偏实用干货型
- 方案B：偏情感共鸣型
分别写开头2句话，我好测试哪个效果好。
```

### 28. 带货转化优化
```
我写了一篇小红书笔记推荐【你的产品】，帮我优化正文。
目标是让人看完想买，但不要像硬广。改完给我对比。
下面是我的原文：【粘贴你的草稿】
```

### 29. 账号定位梳理
```
我计划做一个小红书账号，方向是【你的方向】。
帮我梳理：
- 账号定位（一句话说清楚）
- 3个内容方向
- 每周发几篇、发什么类型
- 差异点（凭什么关注我）
```

### 30. 数据复盘模板
```
帮我设计一个小红书笔记的数据复盘模板。
包含：
- 这篇的标题/发布时间/数据
- 做对了什么
- 做错了什么
- 下篇怎么改
```
